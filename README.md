## 🚀 Get Running in 60 Seconds

```bash
# Clone
git clone https://github.com/mamoor123/Planetary-Pulse.git
cd Planetary-Pulse

# Install
npm install

# Configure (optional — works without any keys!)
cp .env.example .env

# Launch
npm start

# Open 🌍
open http://localhost:3000
```

> 💡 **Zero config required.** The app ships with realistic mock data for all 7 climate datasets. Add API keys to `.env` to unlock live Snowflake, Gemini, Backboard, Auth0, and Solana integrations.
