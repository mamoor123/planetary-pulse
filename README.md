### 🌎 Real-Time Earth Health Dashboard

**See the planet. Understand it. Act on it.**

A full-stack Node.js application combining **six prize-category technologies** into one cohesive platform for environmental monitoring, AI-powered climate insights, personal carbon tracking, and blockchain-verified carbon credits.

## 🚀 Quick Start

```bash
# 1️⃣  Clone the repo
git clone https://github.com/mamoor123/planetary-pulse.git
cd planetary-pulse

# 2️⃣  Install dependencies
npm install

# 3️⃣  Configure environment (optional — works without API keys!)
cp .env.example .env

# 4️⃣  Start the server
npm start

# 5️⃣  Open the dashboard
open http://localhost:3000
```

> 💡 **No API keys needed!** The app works fully with realistic mock data. Add keys to `.env` to unlock real integrations.

### 📊 Interactive Dashboard
- **Global Health Index** — Composite 67/100 score with animated SVG ring
- **7 Environmental Metrics** — Temperature, CO₂, sea level, Arctic ice, deforestation, renewables, biodiversity
- **Animated Globe** — Pure CSS rotating Earth with continents, clouds, and pulse rings
- **Trend Charts** — Switch between 4 datasets with interactive tooltips
- **AI Insights** — 6 climate intelligence cards (critical, opportunity, progress, facts)
- **Action Tracker** — 8 clickable climate actions that boost your health score

### 🤖 AI Climate Assistant (Backboard)
- Persistent conversation threads across sessions
- Automatic fact extraction and memory storage
- Climate-specific knowledge base with semantic search
- Falls back to intelligent mock responses when offline

### 🧮 Carbon Footprint Calculator
- **Transport**: 8 commute modes (car, EV, bus, train, subway, bike, walk, flight)
- **Diet**: 5 diet types with IPCC-based emission factors
- **Energy**: 5 grid types + 3 home sizes
- **Consumption**: 3 levels from minimal to high
- **Smart Recommendations**: Prioritized by impact, sorted by CO₂ reduction potential

### 🪙 Carbon Credit Marketplace (Solana)
- Browse 4 credit types: reforestation, wind, mangroves, direct air capture
- One-click retirement with on-chain transaction signature
- Portfolio tracking with Solana Explorer verification links
- Environmental impact equivalencies (trees, miles, flight hours)

## 🛠️ Technology Integrations

<details>
<summary><b>🟣 Backboard — Persistent AI Memory</b></summary>

### What It Does
Backboard provides stateful API infrastructure for AI applications — persistent conversation threads, automatic fact extraction, and memory that spans across sessions.

### How We Use It
```
Assistant Creation → Thread Management → Message Flow
      ↓                    ↓                 ↓
  Climate Specialist   Session State    Auto Memory
      ↓                    ↓                 ↓
  Custom System Prompt  Context Carry    Fact Extraction
```

### Key Endpoints
| Method | Endpoint | Description |
|--------|----------|-------------|
| `POST` | `/api/backboard/chat` | Chat with persistent AI assistant |
| `GET` | `/api/backboard/memories` | Search climate knowledge base |
| `POST` | `/api/backboard/memory` | Store new climate insight |
| `GET` | `/api/backboard/status` | Connection status |

### Code Snippet
```javascript
// Create a climate-specialized assistant
const assistant = await backboardFetch('/assistants', {
  method: 'POST',
  body: JSON.stringify({
    name: 'Planetary Pulse Climate Assistant',
    system_prompt: 'You are a climate science expert...',
    model: 'google/gemini-2.0-flash',
  }),
});
```

</details>

<details>
<summary><b>🟠 Auth0 for Agents — User + Agent Authentication</b></summary>

### What It Does
Secures user sessions and mints scoped tokens for AI agent access — enabling the pattern where users authenticate once, then delegate specific capabilities to the AI.

### How We Use It
```
User Login (Auth0) → Session Cookie → Agent Token Endpoint
       ↓                                    ↓
  OIDC Flow                            Scoped Token
       ↓                                    ↓
  User Profile                         AI Agent Auth
```

### Key Endpoints
| Method | Endpoint | Description |
|--------|----------|-------------|
| `GET` | `/login` | Auth0 login page |
| `GET` | `/logout` | Auth0 logout |
| `GET` | `/callback` | OAuth callback |
| `GET` | `/api/auth/status` | Auth status + user profile |
| `GET` | `/api/auth/agent-token` | Mint agent authorization token |

### Code Snippet
```javascript
app.use(auth({
  authRequired: false,
  auth0Logout: true,
  clientID: process.env.AUTH0_CLIENT_ID,
  issuerBaseURL: `https://${process.env.AUTH0_DOMAIN}`,
  authorizationParams: {
    scope: 'openid profile email',
    audience: process.env.AUTH0_AUDIENCE,
  },
}));
```

</details>

<details>
<summary><b>🔵 Google Gemini — AI Climate Analysis</b></summary>

### What It Does
Generates personalized climate insights, data interpretations, and action plans using Gemini's generative AI capabilities.

### How We Use It
```
User Query → Structured Prompt → Gemini API → JSON Response
                                            ↓
                                    Parse & Render
                                            ↓
                                    Dashboard Display
```

### Key Endpoints
| Method | Endpoint | Description |
|--------|----------|-------------|
| `POST` | `/api/gemini/analyze` | Analyze environmental metrics |
| `POST` | `/api/gemini/personal-plan` | Generate personalized action plan |
| `POST` | `/api/gemini/interpret-data` | Human-readable data interpretation |

### Code Snippet
```javascript
const response = await fetch(`${GEMINI_URL}?key=${GEMINI_API_KEY}`, {
  method: 'POST',
  body: JSON.stringify({
    contents: [{ parts: [{ text: analysisPrompt }] }],
    generationConfig: { temperature: 0.7, maxOutputTokens: 2048 },
  }),
});
```

</details>

<details>
<summary><b>🔷 Snowflake — Climate Data Warehouse</b></summary>

### What It Does
Provides access to large-scale climate datasets through parameterized SQL queries with year-range filtering.

### Datasets Available
| Dataset | Source | Unit | Range |
|---------|--------|------|-------|
| 🌡️ Global Temperature | NASA GISS | °C anomaly | 1880–2024 |
| 💨 CO₂ Concentration | NOAA / Mauna Loa | ppm | 1960–2024 |
| 🌊 Sea Level Rise | NASA / NOAA | mm | 1993–2024 |
| 🧊 Arctic Sea Ice | NSIDC | million km² | 1980–2024 |
| 🌳 Deforestation | Global Forest Watch | Mha/year | 2001–2023 |
| ⚡ Renewables | IRENA | GW | 2010–2024 |
| 🦋 Biodiversity | WWF / ZSL | Index | 1970–2020 |

### Key Endpoints
| Method | Endpoint | Description |
|--------|----------|-------------|
| `GET` | `/api/snowflake/dashboard` | Aggregated dashboard data |
| `GET` | `/api/snowflake/datasets` | List all datasets |
| `GET` | `/api/snowflake/query/:id?fromYear=2000` | Query specific dataset |

### Code Snippet
```javascript
let sql = `SELECT year, value FROM ${dataset}_data WHERE 1=1`;
if (fromYear) sql += ` AND year >= ?`;
if (toYear) sql += ` AND year <= ?`;
sql += ` ORDER BY year`;
```

</details>

<details>
<summary><b>🟣 Solana — Carbon Credit Blockchain</b></summary>

### What It Does
Tracks carbon offset credits as on-chain tokens with transparent, verifiable retirement records on Solana blockchain.

### How We Use It
```
Browse Credits → Select & Purchase → Retire (Burn)
                                       ↓
                              TX Signature Generated
                                       ↓
                              Solana Explorer Link
```

### Credit Types Available
| Type | Description | Example Project |
|------|-------------|-----------------|
| 🌳 Reforestation | Tree planting & forest conservation | Amazon Rainforest |
| ⚡ Renewable Energy | Wind & solar carbon offsets | Texas Wind Farm |
| 🌊 Blue Carbon | Mangrove & coastal restoration | Indonesia Mangrove |
| 🏭 Direct Air Capture | CO₂ removal from atmosphere | Climeworks Iceland |

### Key Endpoints
| Method | Endpoint | Description |
|--------|----------|-------------|
| `GET` | `/api/solana/credits` | Browse available credits |
| `GET` | `/api/solana/portfolio` | User's credit portfolio |
| `POST` | `/api/solana/retire` | Retire a carbon credit |
| `GET` | `/api/solana/impact` | Environmental impact calculation |

### Code Snippet
```javascript
// Retire a carbon credit (burn on-chain)
const burnTx = new Transaction().add(
  createBurnInstruction(
    creditTokenAccount, CREDIT_MINT,
    keypair.publicKey, tonnes * 1e9
  )
);
const signature = await connection.sendTransaction(burnTx, [keypair]);
```
